# 许可证选择
 
Bytehub+ 定位是去中心化可信基础设施！所谓去中心化就是有多个利益不一致的用户参与组成的网络，换句话说他们是缺乏信任的。在不信任的主体之间建立信用必须采用可信的技术，因此必须是开源的。但是，如果大家使用不同的标准与接口，那么相互仍然不能相互通信，就更别提信任了。因此，我们不希望社区分裂，这种情况在区块链已经发生。因此，对于我们的主要项目 **Fusion** 我们选用GNU Affero Gerneral Public License（AGPL），这是自由软件基金会发布的一个著佐权（copyleft）许可证。该许可证的具体条款可以参考 **Fusion** 项目下的 **LICENSE** 文件，或者到参考自由软件基金会的[官方文档](https://www.gnu.org/licenses/agpl-3.0.txt)。

对于社区的其他项目，则本着使其最为有用，最能重复使用的原则选用适合的许可证。
